## 1. Project Title
Customer Churn Prediction using Machine Learning

---

## 2. Description
Customer churn refers to the situation where customers stop using a company's product or service. Identifying customers who are likely to churn is important for businesses because retaining existing customers is usually more cost-effective than acquiring new ones.

This project predicts customer churn using machine learning techniques and a rule-based system. The dataset contains customer demographic and behavioral features such as age, income, spending score, membership level, and purchase frequency.

The workflow includes data preprocessing, exploratory data analysis (EDA), training machine learning models, comparing their performance, and implementing a rule-based churn prediction system derived from insights obtained during EDA.

The main objective of this project is to analyze customer behavior and determine the factors that influence churn.

### 4\. Usage

The notebook demonstrates the complete churn prediction workflow:

1\. Load the dataset
2\. Clean and preprocess the dataset
3\. Convert categorical features to numerical format
4\. Perform Exploratory Data Analysis (EDA)
5\. Train machine learning models to predict churn
6\. Compare the performance of different models
7\. Implement a rule-based churn prediction system
8\. Analyze feature importance and model results

Machine Learning Models Used

\- Logistic Regression
\- Random Forest
\- Gradient Boosting
\- XGBoost

Model Evaluation
Models are evaluated using:

\- Accuracy
\- Precision
\- Recall
\- F1 Score

The performance of each model is compared to determine which model predicts churn most effectively.
